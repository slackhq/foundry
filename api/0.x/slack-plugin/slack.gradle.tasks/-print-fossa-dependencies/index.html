PrintFossaDependencies

@CacheableTask
abstract class PrintFossaDependencies : BaseDependencyCheckTask(source)

A task that writes runtime dependency info found in identifiersToVersions.

This is used by the Fossa tool to parse and look up our dependencies. The output file is in the form of a newline-delimited list of <module identifier>:<version>.

Example:

mvn+com.google.gson:gson:2.8.6
mvn+com.google.guava:guava:29-jre
