forceAndroidXComposeCompilerForComposeMultiplatform

val forceAndroidXComposeCompilerForComposeMultiplatform: Boolean

If true, uses the AndroidX compose compiler SlackVersions.composeCompiler for Compose Multiplatform compilations rather than the Jetbrains one. This can be useful in testing where AndroidX's compiler is farther ahead.
